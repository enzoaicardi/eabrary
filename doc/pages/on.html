. on

Permet d'attacher un événement à un élément du DOM.

Syntaxe

ea('selecteur').on(event, function, once);

Arguments

event :string

Définit l'événement javascript à associer au selecteur : "click", "touchstart", "wheel", etc...

function :function

Définit la fonction à executer lors du déclenchement de l'évènement : function() { // Do something... } ou () => { // Do something... }

once :boolean //optionnel

Définit si l'événement doit se déclencher seulement une fois.

Exemple

click on the box event

not clicked

onclick : replace text content by clicked
