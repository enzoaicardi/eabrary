. interval

Permet d'ajouter un interval d'execution pour chaque élément du DOM.

Syntaxe

ea('selecteur').interval(duration);

Arguments

duration :number

Définit le l'interval d'execution pour tout les éléments du DOM ciblés par la méthode qui suit, attention car les méthodes suivantes sont affectées par cet interval dans le cas d'un chaînage de propriétés.
Cet argument peut prendre la valeur 'reset' ou 0 pour annuler l'interval définit dans le cas d'un chaînage.

Exemple

click on the box event

text 1

text 2

text 3

set background-color red every 2000ms
