. intersec

Permet d'attacher un événement à un élément du DOM.

Syntaxe

ea('selecteur').intersec(parent, ratio, fonction, side);

Arguments

parent :string

Définit l'élément parent avec lequel le selecteur doit se croiser.

ratio :number

Compris entre 0 et 1 le ratio représente le pourcentage de visibilité du selecteur necessaire au déclenchement de la fonction.

fonction :function

Définit la fonction à executer lors du déclenchement de l'évènement : function() { // Do something... } ou () => { // Do something... }

side :string //optionnel

Définit si l'événement doit se déclencher lors d'un croisement sur le "top", "bottom", "left" ou "right" de l'élément parent.

Exemple

scroll the box event

scroll

set background-color blue when intersec bottom of sandbox with 80% visibility
