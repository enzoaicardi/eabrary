letterbyletter

Permet de transformer un texte en suite de lettres avec chacune une class.

Syntaxe

letterbyletter(selector, class);

Arguments

selector :string

L'élément querySelector à selectionner dans le DOM (un seul élément possible).

class :string

Définit le nom de la class des lettres. Pour les animer n'oubliez pas de définir l'état display.
