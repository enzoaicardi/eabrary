. ea ('selecteur')

Selectionne un élément du DOM.

Syntaxe

ea('selecteur', entrée || 'entrées')

Arguments

selecteur :string

Chemin css (querySelector) correspondant à l'élément cible.

entrée :number //optionnel

Si aucune entrée n'est définie le selecteur pointra vers toutes les entrées correspondantes.
Si une entrée de type :number est définie le selecteur pointra vers l'entrée correpondante (les entrées commencent par 0).

entrées :string //optionnel

Si une entrée de type "0, 1, 4" est définie le selecteur pointra vers les entrées correpondantes 0, 1, 4 en ignorant 2 et 3.
Si une entrée de type "3 > 5" est définie le selecteur pointra vers les entrées comprises entre 3 et 5 en incluant les bornes (ici 3 et 5).
