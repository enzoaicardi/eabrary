. spaceadd

Permet de d'ajouter une propriété css à un élément du DOM.

Syntaxe

ea('selecteur').spaceadd(property, inside-property, value);

Arguments

property :string

Définit la propriété css à ajouter pour l'élément selectionné.

inside-property :string

Définit la propriété css interne à ajouter pour l'élément selectionné.

value :string

Définit la valeur de la propriété.

Exemple

Spaceadd peut être utilisée pour ajouter une propriété filter :

// CSS de départ : filter: opacity(0.5) blur(5px)

ea('selecteur').spaceadd('filter', 'invert', '1');

// CSS d'arrivée : filter: opacity(0.5) blur(5px) invert(1)
