. delay

Permet d'ajouter du delais à la méthode qui suit.

Syntaxe

ea('selecteur').delay(duration);

Arguments

duration :number

Définit le temps d'attente avant l'execution de la méthode qui suit, attention car les méthodes suivantes sont affectées par cet interval dans le cas d'un chaînage de propriétés.
Cet argument peut prendre la valeur 'reset' pour annuler le delais dans le cas d'un chaînage. Cependant cette valeur n'est presque jamais requise si votre code est bien organisé, nous ne conseillons donc pas son utilisation.

Exemple

click on the box event

background

set background-color black after 2000ms
