. animate

Permet de créer une animation complexe sur un ou plusieurs éléments du DOM.

Syntaxe

ea('selecteur').animate(array, wait-the-end);

Arguments

array :array

Il s'agit de spécifier un tableau de propriétés et de valeurs pour créer une animation complexe rapidement. Nous conseillons l'utilisation de cette méthode plutôt qu'un enchaînement de set, add, delay... Cette méthode est plus sûre et performante.

Voir la section ci-dessous pour plus d'informations.

wait-the-end :boolean //optionnel

Spécifiez "true" si vous souhaité que chaque animation se déclanche après celle d'avant.

Array

Le tableau peut être spécifié au préalable et appelé via une variable ou être directement passé dans la chaîne d'arguments de la méthode animate.
Le tableau se rédige de cette manière :

[

 durée(ms),{

   propriété: 'valeur', 
   propriété: 'valeur'

 },'timing-function',

 durée(ms),{

   propriété: 'valeur',
   propriété: 'valeur'

 }

]

Ainsi dans l'exemple ci-dessous :

[

 1000,{

   backgroundColor: 'red', 
   delay: 1000

 },

 500,{

   backgroundColor: 'blue'

 }, 'ease'

]

On commence par un changement de couleur de fond à 'red' qui prendra 1s. Ensuite on ajoute un delais de 1s pour la prochaine animation.
Une fois ce temps écoulé on change la courbe de bézier de 'linear' à 'ease', puis on on oppére un chagement la couleur de fond à 'blue' qui prendra 0.5s.

Mode sans-délais

Il est aussi possible d'activer le mode "nodelay" comme ceci :

[

 'nodelay'

 1000,{

   backgroundColor: 'red', 
   delay: 1000

 },

 500,{

   backgroundColor: 'blue'

 }, 'ease'

]

Ainsi toutes les animations se produiront en même temps avec leur propre timing et fonction, seule la propriété "delay" permet d'ajouter du delais à toutes les animations qui suivent.
