. cssbezier

Permet de changer la propriété css transition-timing-function d'un élément du DOM.

Syntaxe

ea('selecteur').cssbezier(property);

Arguments

property :string

Définit la propriété transition-timing-function de l'élément. Par exemple 'cubic-bezier(.13,1.69,.44,.12)'

Exemple

click on the box event

background

set cssbezier to cubic-bezier(.13,1.69,.44,.12)
