. replace

Permet de d'enlever une valeur à une propriété css d'un élément du DOM.

Syntaxe

ea('selecteur').replace(property, value, new-value);

Arguments

property :string

Définit la propriété css à laquelle remplacer une valeur pour l'élément selectionné.

value :string

Définit la valeur de la propriété à remplacer.

new-value :string

Définit la valeur de remplacement.

Exemple

Replace peut être utilisée pour remplacer une propriété de transition comme ci-dessous :

// CSS de départ : transition: padding-right 2s linear

ea('selecteur').replace('transition', 'padding-right 2s linear', 'margin-top 1s ease');

// CSS d'arrivée : transition: margin-top 1s ease
