. include

Permet d'inclure un fichier dans un élément du DOM.

Syntaxe

ea('selecteur').include(url, error404-url);

Arguments

url :string

Définit le chemin de l'élément à inclure.

error404-url :string //optionnel

Définit le chemin de l'élément à inclure en cas d'erreur 404.

Exemple

La documentation Eabrary se base sur cette méthode.
