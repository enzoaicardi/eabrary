. remove

Permet de d'enlever une valeur à une propriété css d'un élément du DOM.

Syntaxe

ea('selecteur').remove(property, value);

Arguments

property :string

Définit la propriété css à laquelle retirer une valeur pour l'élément selectionné.

value :string

Définit la valeur à retirer à la propriété.

Exemple

Remove peut être utilisée pour retirer une propriété de transition comme ci-dessous :

// CSS de départ : transition: padding-right 2s linear, margin-top 1s ease

ea('selecteur').remove('transition', 'margin-top 1s ease');

// CSS d'arrivée : transition: padding-right 2s linear
