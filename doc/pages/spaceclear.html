. spaceclear

Permet de supprimer une propriété css à un élément du DOM.

Syntaxe

ea('selecteur').spaceclear(property, value);

Arguments

property :string

Définit la propriété css à supprimer pour l'élément selectionné.

value :string

Définit la valeur de la propriété.

Exemple

Spaceclear peut être utilisée pour supprimer une propriété filter :

// CSS de départ : filter: opacity(0.5) blur(5px) invert(1)

ea('selecteur').spaceclear('filter', 'invert');

// CSS d'arrivée : filter: opacity(0.5) blur(5px)
