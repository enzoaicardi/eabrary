. add

Permet de d'ajouter une valeur à une propriété css d'un élément du DOM.

Syntaxe

ea('selecteur').add(property, value);

Arguments

property :string

Définit la propriété css à laquelle ajouter une valeur pour l'élément selectionné.

value :string

Définit la valeur à ajouter à la propriété.

Exemple

Add peut être utilisée pour ajouter une propriété de transition comme ci-dessous :

// CSS de départ : transition: padding-right 2s linear

ea('selecteur').add('transition', 'margin-top 1s ease');

// CSS d'arrivée : transition: padding-right 2s linear, margin-top 1s ease
