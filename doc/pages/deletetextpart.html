. deleteTextPart

Permet de masquer une partie de texte ou inline-block.

Syntaxe

ea('selecteur').deleteTextPart(max-width, inside-timeout, animation-duration, reverse);

Arguments

max-width :string

Définit taille maximale des élements à invisibiliser.

inside-timeout :number

Définit le temps entre les animation des propriétés max-width et color.

animation-duration :number

Définit la durée de l'animation.

reverse :boolean //optionnel

Définit si l'animation doit s'inverser si l'événement se déclenche plus d'une fois, sinon il ne se passera rien.

HTML / CSS

La structure suivante est necessaire pour permettre la création du deleteTextPart.
Les balises "p" et "span" peuvent être remplacées par n'importe quelles balises de type inline ou inline-block.

<p>
 some text here
 <span class="custom">text who disappear</span>
 some text here
</p>

Pas de classes par défaut pour cette méthode.

none

Les éléments suivant sont personnalisables

p
span

Exemple

click on the box event

Delete this now

delete "this", reverse true
