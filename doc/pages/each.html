. each

Créer une boucle qui s'execute pour chaque élément du DOM selectionné.

Syntaxe

ea('selecteur').each(function(item, index){ // Do something... });

Arguments

item, index :arguments

Ils doivent être spécifiés comme arguments de fonction afin de pouvoir être réutilisés dans celle-ci, ainsi item correspond à l'élément du DOM sélectionné et index à son index de tableau.

Exemple

click on the box event

text 1

text 2

text 3

set each backgrounds to red
