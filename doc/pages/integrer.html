Intégrer Eabrary à votre document HTML

Pour intégrer Eabrary et disposer de toutes ses fonctionnalités copiez le code correspondant à la version que vous désirez avant tout autre script (hors librairies) dans votre document HTML.

Dernière version

<script src='https://unpkg.com/eabrary/eabrary.min.js'></script>

Vous pouvez garder votre version de la librairie à jour grace à cette balise, cependant vous exposez votre code à des risques d'incompatibilité, par exemple si un bug nouveau apparaît ou si une fonctionnalité que vous utilisiez vient à être supprimée.
Voir la documentation UNPKG pour plus d'information concernant le versionning.

Utilisez cette balise si vous souhaitez obtenir la dernière version sans être affecté par les bugs (les mises à jour seront moins fréquentes).

<script src='https://unpkg.com/eabrary/stable/eabrary.min.js'></script>
