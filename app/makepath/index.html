Aide : modes de tracé

Les bouttons et touches du clavier  M L C Q Permettent respectivement de changer pour les modes de tracé suivants :

M MoveTo : place les coordonnées au point spécifié sans tracer de ligne.

L LineTo : place les coordonnées au point spécifié et trace une ligne droite depuis le point précédent.

C CurveTo : place les coordonnées au point spécifié et trace une courbe de bézier depuis le point précédent.

Q Quadratic : place les coordonnées au point spécifié et trace une courbe de quadratique depuis le point précédent.

Le boutton ou la touche H permettent de masquer les points et ainsi pouvoir fermer la courbe.

Aide : les clics

 Les différents clics de la souris ont des fonctions diverses.

 Le clic gauche : permet de poser un point sur la zone de tracé, mais permet aussi de déplacer un point en restant appuyé sur ce dernier. Un double clic sur un point entraine sa suppression.

 Le clic droit : permet de poser un point après le dernier point selectionné (pour selectionner un point il suffit de faire un clic gauche sur celui-ci).

 Le clic molette : permet de poser un point avant le dernier point sélectionné.

Aide : les sauvegardes

 Permet d'enregistrer le projet en sauvegarde locale sous le nom spécifié dans le champ texte qui suit (par défaut clean-path-1). Faire deux sauvegardes sous le même nom revient à écraser la première.

S ou  Permettent d'afficher la liste des sauvegardes locales. Un double clic sur la sauvegarde permet de la charger sur le document.

F5 Permet de recharger la page pour obtenir un document vierge mais ne supprimer pas les sauvegardes locales.

 Permet d'obtenir le code SVG du document, de télécharger le fichier SVG mais aussi de télécharger le fichier SCP qui peut ensuite être réutilisé par l'application.

 Permet de choisir une sauvegarde au format SCP uniquement pour l'ouvrir dans l'application.

Aide : les attributs

 ou Enter Permettent d'appliquer les attributs spécifiés dans le champ texte qui suit.

Pour remplir le champ texte et ajouter des attributs il est impératif d'utiliser le bon format : / attribut: valeur /

Pour supprimer un attribut il suffit de placer deux points avant ce dernier comme suit : / ..attribut: valeur / et de valider.
